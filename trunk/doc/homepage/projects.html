Rant (a hardware random number generator)

State = idea, pre alpha!

The mdc board is used as it needs 12 VDC. The hardware is finisched. It is from the "XR232 - Echter Zufall und echtes RS232-Protokoll" project but does not use the RS232 part, it just uses the small part and connects it via schmitt trigger to a input pin. Can calculate lottery numbers. Planned: providing random numbers via network. Drawing pictures from the random numbers.
