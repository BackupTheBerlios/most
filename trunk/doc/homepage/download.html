Download

There is no release available.
Check out the latest labelled version or the head, but be aware there is work in progress.

Anonymous(no commit) checkout:
  • connection type: pserver
  • host: cvs.berlios.de
  • repository path: /cvsroot/most
  • user: anonymous
  • module: trunk
type in your shell:

cvs -d:pserver:anonymous@cvs.berlios.de:/cvsroot/most login

cvs -d:pserver:anonymous@cvs.berlios.de:/cvsroot/most co -P trunk
or
cvs -d:pserver:anonymous@cvs.berlios.de:/cvsroot/most co -P -r label trunk

Registered developer checkout:
  • connection type: extssh
  • host: cvs.berlios.de
  • repository path: /cvsroot/most
  • user: loginname
  • module: trunk
type in your shell:

cvs -dloginname@cvs.berlios.de:/cvsroot/most co -P trunk
or
cvs -dloginname@cvs.berlios.de:/cvsroot/most co -P -r label trunk

For more information read the Berlios_CVS_Howto which you will find on the BerliOS site.
