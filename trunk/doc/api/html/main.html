Software Structure


Applications are build within a board support package and linked with the system and user libraries. The Applications are placed in the "most_root/bsp/board_name/usr" directory. The builded binary files are placed in the application directory.
User Libraries are providing additional functionality and may depend on the system libraries. They are placed in the "most_root/usr" directory. Board Support Packages are providing board setup which is common to all applications for a board. They are using the system libraries and are placed in the "most_root/bsp/board_name/sys" directory.
System Libraries are providing the basic system functionality. They are placed in the "most_root/sys" directory.
